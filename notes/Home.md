Welcome. This is a personal note garden built in the spirit of [Andy Matuschak's evergreen notes](https://notes.andymatuschak.org/).

The idea is simple: notes grow over time, link to each other, and accumulate meaning rather than being discarded after a single use.

This site runs on [semanticdata/evergreen](https://github.com/semanticdata/evergreen), a React implementation of the stacked-panels format Matuschak pioneered. Notes live as plain Markdown files; wiki-links between them form the graph.
